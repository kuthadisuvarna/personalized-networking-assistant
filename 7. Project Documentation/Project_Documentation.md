# Project Documentation

## Project Title
Personalized Networking Assistant

## Description
The Personalized Networking Assistant is an AI-powered web application that helps users generate professional conversation starters for networking events. It analyzes event descriptions using DistilBERT, generates context-aware conversation starters using GPT-2, performs fact checking with the Wikipedia API, and stores conversation history and feedback locally.

## Technologies Used
- Python
- FastAPI
- Streamlit
- DistilBERT
- GPT-2
- Wikipedia API
- JSON
- GitHub

## Features
- Event Theme Detection
- Conversation Starter Generation
- Fact Checking
- History Logging
- Feedback Logging
