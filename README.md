# 🚀 Personalized Networking Assistant

An AI-powered web application that helps users generate smart, personalized conversation starters for professional and social networking events.

---

# 📌 Project Overview

The Personalized Networking Assistant analyzes event descriptions, detects important themes using AI, generates professional conversation starters, performs fact checking using Wikipedia, and stores conversation history and user feedback.

---

# ✨ Features

- Event Theme Detection using DistilBERT
- AI-based Conversation Starter Generation
- Wikipedia Fact Checking
- Conversation History Logging
- Feedback Logging
- FastAPI Backend
- Streamlit Frontend

# ⚙ Installation

```bash
git clone https://github.com/kuthadisuvarna/personalized-networking-assistant.git
cd personalized-networking-assistant
pip install -r requirements.txt
```

---

# ▶ Running the Project

### Start FastAPI

```bash
uvicorn main:app --reload
```

### Start Streamlit

```bash
streamlit run frontend/app.py
```

---

# 📸 Sample Workflow

1. Enter an event description.
2. Detect themes automatically.
3. Generate networking conversation starters.
4. Verify facts using Wikipedia.
5. Store conversation history.
6. Save user feedback.
