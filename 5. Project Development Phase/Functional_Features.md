# Project Development Phase

## Modules Developed

### 1. Event Analyzer
- Extracts important themes from event descriptions using DistilBERT.

### 2. Topic Generator
- Generates professional networking conversation starters using GPT-2.

### 3. Fact Checker
- Retrieves factual information from Wikipedia.

### 4. History Logger
- Stores conversation history in JSON format.

### 5. Feedback Logger
- Stores user feedback for generated conversation starters.

### 6. FastAPI Backend
- Provides API endpoints for frontend communication.

### 7. Streamlit Frontend
- Interactive interface for users to generate conversation starters and perform fact checking.
